Timestamp Microservice

Type in a date after the URL of this page

If what you typed in is either a UNIX timestamp or a natural language date, the page returns both the natural language date and the UNIX timestamp.

If what you typed in is neither a UNIX timestamp nor a natural language date, the page returns 'null' for both properties.

Example

Input: 'https://timestamp.herokuapp.com/8%20October%202017'

Output: {"unix":1507417200,"natural":"October 7, 2017"}
